FAQ

Frequently asked questions

Everything you need to know before getting started.

AI Assistant uses Google Gemini with intelligent fallback across model versions. When no API key is configured, a simulation mode provides sample responses for demo purposes.

Each function includes three pre-engineered prompt structures. Templates control how your input is framed — from expert explanations to concise summaries — so you get consistent, high-quality output.

The Friendly persona uses warm, approachable language ideal for beginners. The Academic persona delivers formal, structured responses suited for research, essays, and professional use.

Queries are sent to the Gemini API for processing. Feedback ratings are logged locally for analytics. No personal account data is collected — the tool is designed for open, friction-free access.
